There are 2 ways to create a new storyboard. Which would you like?

Based on a script

If you are working on a story with a script, you can specify the script, and Storyboarder will help you board for each scene and show you the script while you are boarding. Feel free to change the script while boarding. Scripts can be in Fountain (.fountain) or Final Draft (.fdx) format.

Blank Storyboard

You can simply create a totally blank storyboard. Just make boards and draw. If you'd like to import your storyboard into a script based one later, it's easy.

If you are just starting, why not create a new blank storyboard for fun?

The aspect ratio defines the size of your boards.

2.39:1 is the widest, like what you would watch in a movie.
16:9 is what you would watch on a modern TV.
4:3 is what your grandpops watched back when screens flickered and programming was wholesome.
